CSV Format

Required columns:

Column Name Description Example
receipt_no Unique receipt number GIT/R/25/001
invoice_number Invoice number to link receipt to (e.g., invoice_no from invoice record) GIT/B/001
receipt_date Receipt date (YYYY-MM-DD format) 2025-04-02
amount_received Amount received in the receipt 5000
payment_mode Payment mode (cash, upi, bank_transfer, or card) - optional, defaults to cash upi
notes Notes field for UTR/Transaction number - optional UTR123456789
Important Notes:
  • Each receipt must link to an existing invoice using the invoice number
  • Use the invoice_no from the invoices table (e.g., GIT/B/001)
  • Receipt numbers must be unique in the system
  • Multiple receipts can be recorded for the same invoice (for installment payments)
